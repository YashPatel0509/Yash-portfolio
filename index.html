IoT Security Monitoring with AWS, Suricata & Kibana

Built an end-to-end IoT security lab using a Raspberry Pi temperature sensor and AWS IoT Core. Sensor data was published over MQTT to the cloud, mirrored to an EC2 monitoring server, and inspected by Suricata with Filebeat, Elasticsearch, and Kibana dashboards. Simulated DDoS and malformed MQTT payload attacks and detected them using custom IDS rules and VPC Flow Log analysis.

Tools: Raspberry Pi, MCP9808, AWS IoT Core, Mosquitto, Suricata, Filebeat, Elasticsearch, Kibana, VPC Flow Logs

Part 1 – Device & AWS Setup Part 2 – Python Script Part 3 – Suricata & Kibana Part 4 – VPC Flow Logs

Performed a structured penetration test against the Metasploitable2 vulnerable VM using Kali Linux. Enumerated services, exploited UnrealIRCd, NFS misconfiguration, DistCC remote code execution, and DVWA OS command injection, obtained reverse shells, and documented impact and remediation mapped to OWASP risks.

Tools: Kali Linux, Nmap, Metasploit, Netcat, DVWA

Active Directory Network Vulnerability Assessment Lab

Designed a small corporate-style lab with a Kali attacker VM, a Windows Server 2019 domain controller, and a Windows 10 domain-joined client. Configured Active Directory, verified connectivity, and then performed scanning, vulnerability assessment, exploitation, post-exploitation, and lateral movement across the environment.

Used Nmap, Enum4linux, Nikto, OpenVAS, and Metasploit to move through the full penetration testing lifecycle and finished with remediation recommendations to harden the domain and hosts.

Tools: Kali Linux, Windows Server 2019, Windows 10, Nmap, Enum4linux, Nikto, OpenVAS, Metasploit, Active Directory
